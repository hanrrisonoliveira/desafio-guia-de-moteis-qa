# Desafio Técnico - Guia de Motéis

## Descrição
Este repositório contém o código para o desafio do Processo Seletivo de Engenheiro de QA - Pleno. O projeto é dividido em três partes para demonstrar as habilidades de testes e automação nas seguintes áreas:

- Testes de UI (Interface) com **Cypress**
- Testes de API com **Jest**
- Testes de Performance com **K6**

## Estrutura do Repositório
- **Pasta: Testes de UI (Interface) - Cypress**  
  Contém os testes de interface utilizando o Cypress, uma ferramenta popular para testes end-to-end de UI.
  
- **Pasta: Testes de API - Jest**  
  Contém os testes de API utilizando o Jest, um framework de testes para JavaScript com foco em simplicidade.

- **Pasta: Testes de Performance - K6**  
  Contém os testes de performance utilizando o K6, uma ferramenta para automação de testes de carga e desempenho.

## Tecnologias Utilizadas
- **Cypress**: Para testes de UI e automação de interações com o usuário.
- **Jest + Supertest**: Para testes de APIs e validação de endpoints.
- **K6**: Para testes de performance e análise de carga.
